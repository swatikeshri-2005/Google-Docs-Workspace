# 📝 Google Docs Workspace Clone

A real-time collaborative document editing platform inspired by Google Docs.
Built with modern full-stack technologies to support live collaboration, authentication, document sharing, and cloud-based editing.

---

# 🚀 Features

## ✨ Core Features

* 📝 Create, edit, and delete documents
* ⚡ Real-time collaborative editing
* 👥 Multi-user document sharing
* 🔒 User authentication & authorization
* ☁️ Cloud document storage
* 📄 Rich text editor
* 💾 Auto-save functionality
* 🌙 Responsive modern UI
* 🔍 Search documents
* 📂 Dashboard workspace

---

# 🛠️ Tech Stack

## Frontend

* React.js
* Tailwind CSS
* Socket.io Client
* Quill.js / EditorJS

## Backend

* Node.js
* Express.js
* Socket.io

## Database

* MongoDB Atlas

## Authentication

* JWT Authentication
* bcrypt.js

## Deployment

* Vercel (Frontend)
* Render / Railway (Backend)

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/swatikeshri-2005/Google-Docs-Workspace.git
```

---

## 2️⃣ Install Frontend Dependencies

```bash
cd client
npm install
```

---

## 3️⃣ Install Backend Dependencies

```bash
cd ../server
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
CLIENT_URL=http://localhost:5173
```

---

# ▶️ Run Project

## Start Backend

```bash
cd server
npm run dev
```

## Start Frontend

```bash
cd client
npm run dev
```
